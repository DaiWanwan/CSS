元素的水平垂直居中

水平居中

  • 行内或类行内元素（比如文本和链接）

    在块级父容器中让行内元素居中，只需使用 text-align: center;；这种方法可以让 inline/inline-block/inline-table/inline/flex等类型的元素实现居中。

  • 块级元素

    可以设置 margin: 0 auto;即可实现水平居中。无论父级容器和块级元素的宽度如何变化，都不会影响块级元素的居中效果。

    若知道元素宽度，可以设置position: absolute; left: 50%; margin-left: -width/2;即可实现水平居中。

    若不知道其宽度，可以设置 position: absolute; left: 50%; transform: translateX(-50%);即可实现水平居中。

  • 多个块级元素

    如果要让多个块级元素在同一水平线上居中，其父元素设置text-align: center，子元素display: inline-block，将子元素设置为行内块级元素。

  • flexbox实现多个块状元素的水平居中

    设置display: flex; justify-content: center;即可实现水平居中

垂直居中

  • 行内或类行内元素（比如文本和链接）

    对于单行行内或者文本元素，只需为它们添加等值的padding-top和padding-bottom就可以实现垂直居中。

    如果因为某些原因我们不能使用padding属性来实现垂直居中（行内元素的padding-top和padding-bottom无作用）， 而且已知文本不会换行，那么就可以让line-height和height相等，从而实现垂直居中。

    对于多行文本，同样可以使用等值padding-top和padding-bottom的方式实现垂直居中。

    对于多行文本，如果你在使用过程中发现这种方法没见效，那么你可以通过 CSS 为文本设置一个类似table-cell的父级容器，然后使用vertical-align属性实现垂直居中。

  • 块级元素

    若知道元素宽度，可以设置position: absolute; top: 50%; margin-top: -height/2;即可实现垂直居中。

    若不知道其宽度，可以设置 position: absolute; top: 50%; transform: translateY(-50%);即可实现垂直居中。

  • flexbox实现多个块状元素的垂直居中

    设置display: flex; flex-direction: column;即可实现垂直居中

  • 伪元素

    在垂直居中的元素上添加伪元素，设置伪元素的高等于父级容器的高，然后为文本添加vertical-align: middle;样式，即可实现垂直居中。

水平且垂直居中

  • 宽高固定元素

    设定父级容器为相对定位的容器，设定子元素绝对定位的位置position: absolute; top: 50%; left: 50%;， 最后使用负向margin实现水平和垂直居中，margin的值为宽高的一半。

  • 宽高不固定元素

    设定父级容器为相对定位的容器，设定子元素绝对定位的位置position: absolute; top: 50%; left: 50%;， 最后使用transform: translate(-50%, -50%);实现水平和垂直居中。 使用transform有一个缺陷，就是当计算结果含有小数时（比如 0.5），会让整个元素看起来是模糊的，一种解决方案就是为父级元素设置 transform-style: preserve-3d;。

  • flexbox

    设置display: flex; flex-direction: column; justify-content: center
